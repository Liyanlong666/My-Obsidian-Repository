### 1. 🔍 题目解构分析

- **核心问题识别**：提取题目的数学本质

- **题型分类标签**：明确所属算法类别### 2. 🧠 多解法思维展开

- **暴力解法**：最直观的解决思路（O(n²)或更高）
- **优化方向**：数据结构优化、算法策略优化、数学性质利用
- **最优解法**：时间/空间复杂度最优方案
- **权衡分析**：不同解法的适用场景对比### 3. 💻 代码实现标准

- **多语言支持**：Python（简洁）、Java（面试标准）、C++（竞赛标准）
- **代码规范**：清晰命名、适度注释、模块化设计
- **错误处理**：输入验证、边界保护
- **可读性优先**：优雅胜过聪明### 4. 📊 性能分析深度

- **时间复杂度**：最好、平均、最坏情况分析
- **空间复杂度**：辅助空间使用情况
- **实际性能**：常数因子、缓存友好性考虑
- **可扩展性**：大数据场景下的表现预估### 5. 🧪 测试策略完备

- **基础功能测试**：题目给出的示例
- **边界测试**：最小/最大输入、空集合
